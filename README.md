360t7使用[Hanwckf的OpenWrt源码](https://github.com/hanwckf/immortalwrt-mt798x)

x86使用[Lean的OpenWrt源码](https://github.com/coolsnowwolf/lede)

ua2f防检测插件[说明](https://github.com/QiYueYiya/OpenWrt-Action/blob/main/ua2f.md)
